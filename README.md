Timur has guessed a number from
1 to n (inclusive), and Ruslan is going to guess it. Each time Ruslan calls a number to Timur, and Timur answers him in one of the ways: "more", "less", "you guessed it!". In what is the least number of attempts Ruslan can be guaranteed to guess Timur's number?

Input data format
The input to the program is a natural number n.

Output data format
The program should output the least number of attempts guaranteed to be enough for Ruslan to guess Timur's number.
